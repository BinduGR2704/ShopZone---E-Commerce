How to run this application??

Terminal 1 --------------------

1. python -m venv venv
2. venv\Scripts\activate

3. pip install flask flask_sqlalchemy pymysql flask-jwt-extended flask-bcrypt flask-cors python-dotenv cryptography

4. cd backend
5. python app.py

Terminal 2 --------------------

1. cd frontend
2. python -m http.server 8080

In browser --------------------

http://localhost:8080/home.html
